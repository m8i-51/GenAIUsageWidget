# Widget Edge Hide (PC Manager style)

## Goal

デスクトップウィジェットを Microsoft PC Manager のフローティングツールバーと同様に、画面端へ吸着させて隠せるようにする。

## Behavior

1. ウィジェットを左右の画面端近くまでドラッグして離すと、その端にドックして **同じモニタ上で** 細い peek タブにリサイズする（画面外へスライドしない＝サブモニタへ飛ばない）。
2. peek にカーソルを乗せると展開する。カーソルが離れ、かつドック近傍から外れたら再折りたたみ（逃げるループ防止）。
3. ヘッダーの Hide ボタンでも、近い左右端へドック＆折りたたみできる。
4. 端から離してドラッグするとドック解除（通常のフローティング表示に戻る）。
5. トレイの「Hide Desktop Widget」は従来どおり完全非表示（別操作）。
6. ドック状態（端）は `settings.json` に永続化し、再起動後も折りたたみ状態で復元する。

## Settings

```json
{
  "widgetEdgeHide": null | "left" | "right"
}
```

`widgetBounds` は展開時（端にぴったり接した位置）を保存する。折りたたみ座標は保存しない。

## Out of scope

- 上下端へのドック（縦長カードのため左右優先）
- 完全非表示のトレイ操作の置き換え
